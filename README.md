# Volley!

An idle pong game that lives on your desktop. A small window, always there, gently bouncing away while you work. The main character has a dream: beat the world volley record. You train, buy items, recruit partners, and inch toward that number one bounce at a time.

Built in [Godot](https://godotengine.org) and developed in the open.

## Play

*The prototype is in active development. Playable builds land on [itch.io](https://volcanoem.itch.io) as milestones hit.*
